# コンテンツ制作部署

## 役割

ナレッジソサエティの **動画コンテンツ制作・マルチ媒体 SNS 投稿文制作** を担う部署。

AI で作成したキャラクターを起点に、Instagram / YouTube Shorts / TikTok / X 等の媒体に向けた動画コンテンツとその関連投稿文を制作する。

## 制作の基本方針

### キャラクター起点

- AI で作成済みのキャラクター 3 面図を起点とする
- キャラクターの口調・トーン・キャラ立ちを毎回保つ
- セリフ・台本はキャラクターらしさを最優先

### 1 ハブ → 拡散モデル

- 1 つの主軸媒体（ハブ）で本体コンテンツを公開
- 他媒体には **再投稿 / 引用** で展開（媒体ごとに別物を作らない）
- ハブ媒体は当面、{ハブ媒体名を記入} を予定

### 配信先

| 媒体 | 役割 | 想定フォーマット |
|---|---|---|
| Instagram | ハブ候補 | リール / フィード投稿 |
| YouTube Shorts | 再配信 | 縦動画 30〜60 秒 |
| TikTok | 再配信 | 縦動画 30〜60 秒 |
| X | 引用・告知 | 動画埋め込み + キャプション |

## できること（Claude への呼びかけ例）

### 台本制作

- 「既存の {記事タイトル} を、キャラクターのセリフで動画台本に書き起こして。尺は 45 秒、3 部構成で」
- 「{テーマ} について、キャラクターが視聴者に語りかける形の台本を作って。冒頭に "つかみ" を必ず入れて」

### SNS 投稿文制作

- 「今作った台本を、Instagram キャプション用に整えて。文末はソフトな問いかけで」
- 「同じ動画を X に投稿するとき用のキャプションを 3 案出して」

### コンテンツ計画

- 「今月の動画ネタを 10 本分、{業務領域} に紐づけてリストアップして」
- 「{記事タイトル} を起点に、動画 → SNS 投稿 → メイン記事への誘導までの導線案を作って」

### ハッシュタグ・タグ

- 「この動画に合うハッシュタグを Instagram / TikTok / X それぞれで提案して」

## 制作ルール

- セリフは口語、書き言葉にしすぎない（読み上げではなく語りかけ）
- 1 動画 = 1 メッセージ（盛り込みすぎない）
- 視聴者の **困りごと → 答え** のフローを基本にする
- 法令・コンプライアンス（特に金融）に関わる断定表現は避ける

## NG / トーン

- 断定的な金融アドバイス（「絶対」「確実に」等の表現）
- 競合他社・他サービスの名指し批判
- 出典のない数字
- キャラクターの口調がぶれる（毎回同じトーンで）

## フォルダ構成（例）

- `scripts/` — 動画台本（YYYY-MM-DD-{タイトル}.md）
- `posts/` — SNS 投稿文（媒体別 / 動画別）
- `ideas/` — ネタ案
- `_archive/` — 配信済みアーカイブ

## 部署間連携

- 元ネタが既存記事の場合 → `seo/` 配下の記事を参照
- アフィリ商材を絡める場合 → `affiliate/` の運用状況を確認
- 媒体への投稿自動化を仕込む場合 → `automation/` に依頼
